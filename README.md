# WhisperOfTheSound
Whisper of the Sound explora la relación entre sonido y materia visual.  Cada interpretación genera un paisaje único en tiempo real: texturas que emergen, se deforman y desaparecen, guiadas por la energía, el ritmo y la memoria del sonido.  La obra propone una experiencia donde el sonido no solo se escucha, sino que se vuelve visible.
